MY CONTRIBUTIONS TO THE PROJECT

Select Piece Sequence Diagram

The Select Piece sequence diagram follows the steps functions that are called when a player clicks the drop-down menu to select a piece to play the game. Then the piece of choice is updated in the system and the player places the piece on the board by selecting the 'place a piece' button. The system makes sure the piece is able to fit in the board and updates the board.

Change Color-Blind Setting Sequence Diagram

The Change Color-Blind Setting sequence diagram shows the sequence of functions that are called when a player selects the color blind mode and chooses a color setting from the main page of the game. The system then updates the colors of the pieces and the board.

Give Up Fully Dressed Use Case and Sequence Diagram

The Give Up use case follows a success scenario when a player gives up playing the game. When the player chooses to exit the game the system displays a message asking the user if they want to save the current progress of the game. If no is selected the game is wiped and then the player can choose to click the 'exit' button and stop playing the game.

The Give Up sequence diagram follows the functions called in order to end the game step by step.
